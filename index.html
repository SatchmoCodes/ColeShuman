Color Fill Game

HTML

CSS

JavaScript

Responsive

For this project, I used vanilla JavaScript to make a color fill game. Starting in the top left corner, the goal is to fill the entire square with the same color in as few turns as possible.

Figuring out the logic for getting each direction to work properly, as well as developing the other features of this game, was a fun and challenging task. Learning and implementing data attributes went a long way in this project.

To play the game, you click any of the 5 colored circles in the bottom to change the top left square, as well as any other squares you have captured. In the 4 circle row with the white buttons, the first resets the board, the second saves the current board when refreshing, the third clears the save, and the fourth shows what squares you will capture in the next move after hovering over the colored circles.

A counter at the top keeps track of your turns. The export and import feature allows you to save the code for a board so you and a friend can play on the same board. There is a high score tracker as well as a counter for each type of square. Finally, there is a menu to change the color of the squares.

This project uses Node.js and MongoDB to create a CRUD application. Using a Python script to extract entries from an Excel sheet, I was able to upload the entries into the database. The site holds the means to create new entries, methods to retrieve the entries, the ability to update entries, and the ability to delete them.

In addition to the languages listed above, EJS and JavaScript code and functions allow the user to search specific dates and entries. There is also a multiple choice game at the bottom of the homepage that randomly grabs four entries and tracks correct answers.
